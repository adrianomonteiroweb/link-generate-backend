# link-generate-backend

Link generator for whatsapp from number and with message.

### Technologies:

- node.js
- express.js
- Joe
- Jest
- Frisby
- heroku

### How to run the application:

```bash
   npm start
```

- How to test the application:

```bash
   npm run test
```

- How to test a specific block:

```bash
   npm run test tests/link-generate.test.js
```
